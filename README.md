# MaxGen
Наша визуальная новелла состоит из двух главных сюжетных линий. Это история персонажа Максимилиана и история Гены. Первый из них идет обучаться профессии C# разработчика в УрФУ, а второй хочет познать эту профессию на онлайн курсах. Игрокам предлагается выбрать, за кого из персонажей они будут проходить нашу новеллу, но советуем узнать историю как Максимилиана, так и Гены, и самим сделать вывод о том, кто в жизни добился большего.
